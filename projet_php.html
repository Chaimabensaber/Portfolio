MINI BLOG NEWS DU WEB

Dans le cadre d'un projet universitaire, nous avions pour objectif de réalise run mini blog d'articles en php avec la possibilités pour ceux qui sont connéctés de poster un commentaire visible par tous.


Afin de pousser un peu plus loin la consigne, j'ai décidé de réaliser un mini blog d'actualités sur les news du web.

Développement Web
