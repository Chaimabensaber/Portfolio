UNE JOURNEE IMPRESSIONNANTE

Dans le cadre d'un projet universitaire de groupe d'une durée de 3 mois, nous avions pour objectif de créer une agence fictive qui s'occupe de la création d'expositions et de musées.


L'exposition du moment se nomme : Une Journée Impressionnante, une exposition sur le thème de l'impressionnisme dont les tableaux prendront place dans l'ordre chronologique d'une journée.

UX / UI

Développement Web

YEAR: 2023

Voir la version en ligne ↗

Dans le cadre de cette exposition, nous avons tout d'abord réaliser des maquettes fonctionnelles sur le logiciel Figma. Afin de les réaliser, nous avons rédiger tous les textes nécessaires, puis nous avons du utiliser la charte graphique fournit.


Après la validation de ces maquettes par l'entièreté du groupe, nous avons pu passer à la création de ce site web avec React.js.
Lors de sa conception, j'ai eu le rôle de développeuse front-end. Mes missions consistaient donc à intégrer la maquette Figma et rendre le site web dynamique.
